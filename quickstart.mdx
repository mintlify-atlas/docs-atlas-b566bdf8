Get started with Rotations by adding your first tracks and creating your initial rotation.

## Prerequisites

Before you begin, make sure you have:
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Music from Bandcamp or a Spotify account

<Info>
Rotations is a web-based music player that runs entirely in your browser. No server-side setup required for the player itself.
</Info>

## Get started with Rotations

<Steps>
  <Step title="Access the player">
    Navigate to the Rotations player in your web browser. The player interface includes:
    - **Main player controls** - Play, pause, seek, and volume
    - **Current rotation view** - See which tracks are in your active rotation
    - **Add music button** - Import tracks from supported sources
  </Step>

  <Step title="Add your first music">
    Click the **Add Music** button to import tracks. Rotations supports multiple sources:
    
    <Tabs>
      <Tab title="Bandcamp">
        1. Navigate to any Bandcamp album or track page
        2. Copy the full URL (e.g., `https://artist.bandcamp.com/album/album-name`)
        3. Paste the URL into Rotations
        4. The scraper will extract all track information and audio streams
        
        ```typescript
        // Rotations automatically detects Bandcamp URLs
        if (url.includes('bandcamp.com')) {
          const response = await fetch('/api/bandcamp/scrape', {
            method: 'POST',
            body: JSON.stringify({ url })
          });
          // Tracks are added to your library
        }
        ```
      </Tab>
      
      <Tab title="Spotify">
        1. Click **Connect Spotify** in the player
        2. Authorize Rotations to access your Spotify library
        3. Import individual playlists or your recently played tracks
        4. Audio streams are fetched from Spotify's API
        
        <Note>
        You'll need a Spotify Premium account to stream full tracks through Rotations.
        </Note>
      </Tab>
    </Tabs>
  </Step>

  <Step title="Your first rotation">
    Once you've added music, it automatically enters your **Current Rotation**:
    
    - The first track you listen to becomes the seed of your rotation
    - Subsequent plays create associations between tracks
    - Your rotation grows organically based on your listening patterns
    
    ```typescript
    // From site/src/lib/associations.ts
    export function updateAssociations(
      fromTrackId: TrackId,
      toTrackId: TrackId,
      playlistId: PlaylistId
    ): void {
      const key = `${fromTrackId}:${toTrackId}`;
      const count = associations.get(key) || 0;
      associations.set(key, count + 1);
      
      // Associations determine your rotation
    }
    ```
  </Step>

  <Step title="Explore your graph">
    As you listen to more music, Rotations builds a **listening graph** that captures your patterns:
    
    1. Click the **Graph View** button to visualize your listening history
    2. Each node represents a song, sized by play count
    3. Edges show transitions between songs
    4. Colors indicate different clusters or genres
    
    The graph uses PageRank to identify your most important tracks - not just the ones you play most, but the ones that connect your listening together.
  </Step>
</Steps>

## What happens when you listen

Every time you play a track in Rotations:

1. **Automatic caching** - The audio is cached in IndexedDB for offline playback
2. **Association tracking** - The transition from the previous track is recorded
3. **Rotation updates** - Your current rotation adjusts based on the new data
4. **Graph evolution** - The listening graph grows with each play

```typescript
// From site/src/lib/store.ts
export class Database {
  setPlayerState(state: Partial<PlayerState>): void {
    if (state.activeTrack !== undefined) {
      store.setValue("activeTrack", state.activeTrack);
      // Triggers association updates and rotation recalculation
    }
  }
}
```

## Understanding rotations

Your **rotation** is the set of tracks that Rotations recommends for your current listening session. Unlike traditional playlists:

- **Dynamic** - Rotations evolve based on what you actually listen to
- **Contextual** - Multiple rotations for different moods or times
- **Graph-based** - Recommendations use your entire listening history, not just recent plays

<Card title="Rotation radius" icon="circle-nodes" href="/features/rotations">

## Troubleshooting

<Accordion title="Music won't play">
  Check that:
  - You have an active internet connection (for first-time streaming)
  - The source URL is correct and accessible
  - For Spotify: You're logged in and have an active Premium subscription
  - Your browser allows autoplay and audio playback
</Accordion>

<Accordion title="Spotify connection fails">
  Make sure you've authorized the Rotations app in your Spotify account settings. If you revoked access, you'll need to reconnect:
  1. Go to your [Spotify Account Settings](https://www.spotify.com/account/apps/)
  2. Remove Rotations if it's listed
  3. Reconnect through the Rotations player
</Accordion>

<Accordion title="Graph view is slow">
  For large libraries (1000+ tracks), the graph visualization can be demanding:
  - Use the search/filter to focus on a subset
  - Try the MDS layout instead of ForceAtlas2
  - Enable GPU acceleration in your browser settings
</Accordion>
